# Plateforme de Pari E-Sportif

Découvrez la plateforme en ligne : https://pari-e-sportif-frontend-vqvv.vercel.app

Vous pouvez également installer et exécuter le projet localement en le clonant sur votre poste de travail.

---

## Technologies utilisées

- **Vue.js** – Framework JavaScript moderne  
- **TypeScript** – Typage statique pour JavaScript  
- **Tailwind CSS** – Framework CSS utilitaire  
- **HTML** – Structure des pages  
- **Fetch API** – Requêtes HTTP

---

## Installation et lancement

1. Cloner le projet :
   
git clone https://github.com/FatoumataDIABATE/Pari_E-Sportif_Frontend.git

cd Pari_E-Sportif_Frontend

npm install

npm run dev
